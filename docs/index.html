What it does

Every feature
you actually need.

#

Nested tags

Organise tasks into #work #backend hierarchies. Tags collapse cleanly, expand on demand.

!

Priorities & due dates

!1 urgent · !2 medium · !3 normal. Due dates highlight red when overdue, yellow when close.

🔒

Optional encryption

AES/Fernet with PBKDF2-HMAC-SHA256 (480k iterations). One command: todo --encrypt.

👤

Per-user isolation

Each OS user has their own task file, stored in a chmod 700 directory. No shared state.

↑↓

Interactive TUI

Full arrow-key navigation, inline editing, and tag management. Works on Linux, macOS, and Windows.

∅

Zero dependencies

One Python file. No frameworks, no daemons, no config. Encryption is opt-in with a single pip install.

Command reference

Everything
in one place.

Adding & viewing

todoShow all tasks (tags collapsed)
todo --allExpand everything
todo -a task textAdd task to root
todo -a #work bugAdd to tag (creates if missing)
todo -a !1 @2026-06-20Priority + due date
todo -s keywordSearch across all tags

Managing & removing

todo -d 3Toggle done on task #3
todo -e 2 new textEdit task #2
todo -m 2 upReorder tasks
todo -r 3Remove task #3
todo --rmtag #workRemove tag + all tasks inside
todo --encryptEnable encryption

Security

Your data.
Your machine.
Nobody else.

No telemetry. No sync. No accounts. Your tasks live in a plain JSON file — or, if you opt in, in an AES-encrypted blob — on your own machine and nowhere else.

  • Linux ~/.local/share/todo/tasks.json
  • macOS ~/.local/share/todo/tasks.json
  • Windows %APPDATA%\todo\tasks.json

Encryption stack

Algorithm:   AES-128-CBC (Fernet)
KDF:         PBKDF2-HMAC-SHA256
Iterations:  480,000 (NIST 2023)
Salt:        32-byte random, per-user
Writes:      Atomic (tmp → rename)
Permissions: chmod 600 (Unix)

Enable in one step

pip install cryptography
todo --encrypt

Install

One command.
Any OS.

One-liner

curl -fsSL https://raw.githubusercontent.com/luuucciiffeerr/todo/main/install.sh | bash

Clone & install

git clone https://github.com/luuucciiffeerr/todo.git cd todo && bash install.sh

The installer checks Python 3.8+, optionally installs cryptography, asks whether to install globally or per-user, and offers to add the bin directory to your PATH.

PowerShell one-liner

irm https://raw.githubusercontent.com/luuucciiffeerr/todo/main/install.ps1 | iex

Clone & install

git clone https://github.com/luuucciiffeerr/todo.git cd todo .\install.ps1

Creates todo.bat and todo.cmd, adds to PATH, supports per-user or global (admin) install.

Anywhere with Python on PATH

# 1. Copy todo.py to a directory on your PATH cp todo.py ~/.local/bin/todo.py # 2. Create a launcher echo '#!/bin/sh' > ~/.local/bin/todo echo 'exec python3 ~/.local/bin/todo.py "$@"' >> ~/.local/bin/todo chmod +x ~/.local/bin/todo
